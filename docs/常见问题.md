# 常见问题与解决方法
## 问题一
**描述：** 报错找不到 ROS包下的PX4
```bash
Resource not found: px4
ROS path [0]=/opt/ros/noetic/share/ros
ROS path [1]=/home/用户名/Sunray/Simulation/sunray_simulator
ROS path [2]=/home/用户名/Sunray/General_Module/sunray_common/sunray_msgs
ROS path [3]=/opt/ros/noetic/share
The traceback for the exception was written to the log file
```
**解决：** 创建链接
```bash
sudo ln -sf ~/sunray_px4 /opt/ros/noetic/share/px4

# 验证 
rospack find px4

# 预期输出
/opt/ros/noetic/share/px4
```
## 问题二
**描述：** 编译功能包时 sunray_msgs 报错
**解决：** 优先编译 sunray_msgs
```bash
# 单独编译
catkin_make --source General_Module/sunray_common/sunray_msgs --build build/sunray_msgs
```

>Tips：若出现类似的报错，可以先单独编译对应的模块。
## 问题三
**描述：** 运行.launch文件时报错找不到相关包，例如
```bash
RLException: [sunray_communication_bridge.launch] is neither a launch file in package [sunray_communication_bridge] nor is [sunray_communication_bridge] a launch file name The traceback for the exception was written to the log file
```
**解决：** 设置包路径
```bash
export ROS_PACKAGE_PATH=$ROS_PACKAGE_PATH:/home/用户名/Sunray/Comunication_Module:/home/用户名/Sunray/General_Module:/用户名/xwms/Sunray/General_Module
```
