# 搭建无人机仿真平台
在开始之前，先说明下无人机的机载电脑中**也有**一个仿真平台（这平台版本比较低，对比官方github仓库少了很多脚本），对于**个人**研究完全足够，但对于**一个团队**还是有必要人手一个仿真平台，方便调试

## 准备工作
1. **Ubuntu 20.04 系统（建议在windows系统开虚拟机）**
2. **安装好 ROS1，确认各个指令可正常执行。** 参考 [ROS的安装](./ROS的安装.md)
## 搭建步骤
### 步骤一
克隆官方仓库：
```bash
git clone https://github.com/YunDrone-Team/Sunray.git -b sunray150
```
安装相关依赖：
```bash
cd Sunray

./install_dependencies.sh
```
### 步骤二
请参考官方文档 [Sunray开源无人机仿真与实践平台](https://wiki.yundrone.cn/docs/sunray)

参考顺序：
1. PX4开发环境配置
2. 无人机控制模块
3. 无人机仿真模块

完成以上步骤后验证是否安装完成：
```bash
roslaunch sunray_simulator sunray_sim_1uav.launch # 能够正常启动gazebo仿真平台
```
**若出现错误，先查看 [常见问题](../常见问题.md) ，若未能解决，再询问AI。**
