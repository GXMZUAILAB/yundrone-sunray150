# 基础网络配置
## 配置仿真环境网络
1. 在ubuntu虚拟机中将网络连接模式改为**桥接模式**，若是**VMware**，再将"复制物理网络连接状态"打勾
2. 进入ubuntu系统中，建议在开发阶段暂时关闭防火墙
```bash
sudo ufw disable
```
3. 确保ubuntu与外界windows处在同一网段（比如 `192.168.x.x` 或 `10.x.x.x`）
```bash
# windows ip 查询
ipconfig

# ubuntu ip 查询
ip a
```
