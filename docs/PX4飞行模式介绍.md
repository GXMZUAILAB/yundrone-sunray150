# PX4飞行模式介绍
## 一、四种模式的核心区别

|模式|控制层级|稳定姿态|定高|定点|外部指令|
|---|---|---|---|---|---|
|**自稳模式**|手动|✅|❌|❌|❌|
|**定高模式**|半自主|✅|✅|❌|❌|
|**定点模式**|半自主|✅|✅|✅|❌|
|**板外模式**|全自主|✅|✅|✅|✅|

## 二、各模式详解

### 1. 自稳模式（Stabilize mode）

**控制方式**：遥控器摇杆直接控制飞机的**姿态角**（俯仰/横滚/偏航），油门控制推力，但**不自动定高**

**适用场景**：
- **新手练习**：熟悉无人机的基本操控手感
- **特技飞行**：翻跟头、快速机动等需要直接姿态控制的操作
- **传感器失效时的手动备降**：当 GPS、光流、激光雷达等传感器不可用时，只能用自稳模式手动降落
- **调试 PID 参数**：测试飞控的姿态响应

**特点**：
- 需要持续手动控制油门来维持高度
- 摇杆回中，飞机会**保持水平**，但会随风飘移
- 最基础、最可靠的模式

---

### 2. 定高模式（Altitude mode）

**控制方式**：飞控**自动维持高度**（通过气压计/激光雷达/超声波），摇杆控制水平方向的速度和姿态。

**适用场景**：
- **航拍/巡检**：保持固定高度，专注于水平移动和构图
- **室内飞行**（无 GPS）：结合定高雷达，可在室内稳定飞行
- **复杂地形飞行**：在山地、建筑群上方飞行时，防止高度突变
- **新手进阶**：从自稳模式过渡到半自主飞行

**特点**：
- 油门摇杆控制**垂直速度**（推杆上升，拉杆下降），松手后悬停当前高度
- 水平方向会飘移（需要手动控制）
- 不依赖 GPS，适合室内或 GPS 信号差的环境

---

### 3. 定点模式（Position mode）

**控制方式**：飞控**同时自动定高和水平位置保持**（需要 GPS 或光流/视觉定位），摇杆控制水平方向的移动速度。

**适用场景**：
- **精准悬停**：航拍、测绘、定点观察时，松开摇杆即悬停在原地
- **航线飞行**：执行预设航点任务时的基础模式
- **自动返航（RTL）**：返航过程中通常使用定点模式
- **激光雷达测绘**：需要飞机在指定位置悬停扫描时

**特点：**
- 摇杆控制的是**水平速度**（前/后/左/右），松手后自动刹停并悬停
- 依赖 GPS 或视觉定位系统（如光流 + 激光雷达）
- 是最常用的半自主飞行模式

---

### 4. 板外模式（Offboard mode）

**控制方式**：飞控完全接收**外部计算机（如机载电脑）** 的控制指令，可以控制位置、速度、姿态、推力等任意组合。

**适用场景**：
- **激光雷达 SLAM 飞行**：机载电脑运行 FAST-LIO2 等算法，实时计算位置并发送给飞控，实现无 GPS 的自主导航
- **路径规划**：机载电脑运行 EGO-Planner 等算法，实时规划避障路径并发送给飞控执行
- **视觉跟踪**：相机识别目标后，机载电脑计算跟踪轨迹并发送控制指令
- **集群编队**：多机协同飞行时，中央控制计算机统一发送指令
- **高级科研**：任何需要机载电脑参与控制决策的场景

**特点**：
- 飞控只负责执行底层控制（电机驱动、传感器融合），所有高层指令来自外部
- 需要持续以 **≥2Hz** 的频率发送指令（通常 20-50Hz），否则超时会自动切回安全模式
- 是无人机自主化、智能化的核心模式

---
## 总结
|模式|核心功能|适合场景|依赖传感器|
|---|---|---|---|
|**自稳**|姿态控制|手动飞行、特技、紧急备降|IMU|
|**定高**|姿态 + 高度控制|室内飞行、定高巡检|IMU + 气压计/激光雷达|
|**定点**|姿态 + 高度 + 水平位置|航点飞行、精准悬停|IMU + 气压计 + GPS/光流|
|**板外**|外部计算机全权控制|SLAM、避障、编队、自主飞行|外部计算机 + 相应传感器|
